# Portfolio  Trần Tiến

Chào mừng! Đây là phiên bản portfolio tĩnh `tvn` của Trần Tiến: nơi trình bày dự án, kỹ năng và thông tin liên hệ.

## Giới thiệu

Mình là Trần Tiến  lập trình viên frontend tập trung vào các giải pháp web tĩnh, trải nghiệm nhẹ và tối ưu. Portfolio này là bản demo để trình diễn các dự án nhỏ và kỹ năng.

## Dự án nổi bật

- Demo trang cá nhân / blog  `index.html`
- Widget / mini-demo JavaScript  `render.js`, `script.js`
- Layout & styles tùy chỉnh  `style.css`

(Ghi chú: cập nhật mô tả cụ thể cho từng dự án, kèm link nếu có.)

## Kỹ năng & Công nghệ

- HTML, CSS, JavaScript (vanilla)
- Responsive UI, tối ưu hiệu năng
- Triển khai: GitHub Pages / Netlify / Vercel

## Chạy & kiểm tra

1. Mở `index.html` trực tiếp trong trình duyệt.
2. Hoặc phục vụ bằng web server tĩnh (khuyến nghị):

```bash
python -m http.server 8000
```

Mở http://localhost:8000 để xem.

## Phát triển

- Chỉnh sửa HTML/CSS/JS trong repo và reload trình duyệt.
- Muốn thêm pipeline (npm, build, gh-pages), tôi có thể hỗ trợ cấu hình.
